# Claude Code Skills

A collection of custom skills for Claude Code that extend its functionality.

## Skills Available

### 1. safe-code-modify
全修改代码，先给计划、经你确认再执行，保留原代码为注释。

This skill:
- Provides a safe way to modify code
- First presents a plan for modification
- Requires user confirmation before execution
- Preserves original code as comments

### 2. safe-delete
安全删除文件，将文件移动到垃圾文件夹而不是直接删除

This skill:
- Safely deletes files by moving them to a trash folder
- Prevents permanent file deletion
- Can be recovered if needed

## Installation

These skills can be installed by placing them in your Claude Code skills directory.
